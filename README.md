# Booking Window

Простое окно бронирования с тремя полями: **Имя**, **Email**, **Телефон** и кнопкой **Забронировать**.  
Форма реализована на HTML/CSS/JavaScript (Angular). Отправляет POST-запрос на тестовый бэкенд.

## Демо
Работающая версия доступна на GitHub Pages:  
[marselchik73.github.io/Booking-window](https://marselchik73.github.io/Booking-window/)

## О проекте
Данный проект создан мной в демонстрационных целях. Он представляет собой прототип окна бронирования, разработанный при поддержке ИИ. Главная цель - наглядно показать процесс покрытия кода тестами: как ручными, так и автоматизированными. Тесты находятся в отдельном репозитории [booking-window-tests](https://github.com/marselchik73/booking-window-tests), а SQL-запросы, используемые для подготовки тестовых данных и проверки результатов, в репозитории [SQL-Examples](https://github.com/marselchik73/SQL-Examples).  
Бэкенд реализован как тестовый, поэтому все запросы обрабатываются без сохранения в реальную базу данных.
